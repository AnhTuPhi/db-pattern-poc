Thirty runnable Spring Boot modules, each reproducing one way a database interaction fails in production — then proving the fix with real SQL, real timing, and the failure you can watch happen. This page is the map: the request flow, every pattern, and what breaks when you scale.

Almost every database failure is invisible in code review. It only surfaces under production concurrency, data volume, or scale — so we made each one runnable, not just described.
